Rear Suspension Redesign

Overview

The goal of this project was to redesign an overbuilt suspension to save mass without sacrificing safety. The previous design was not optimized and used large fasteners (1" diameter) that were expensive and made servicing challening. The goal was to redesign all components with a FoS = 1.25. Although my primary focus for the project was the lower clevises, I participated in the design and review of all the suspension components.
The project can be divided into roughly 8 steps:

  1. Solve for component reaction forces using static setup with loads from competition regulations
  2. Create design
  3. Perform tolerance stackup analysis to ensure smooth-functioning joints
  4. Determine bolt preload required based on fastener sizes and materials
  5. Perform FEA on each fabricated component and validate off-the-shelf components
  6. Perform lug analysis on all lugs
  7. Remove material not along the load path and repeat 3-6
  8. Manufacture and assemble
